Terms of plugin use

This document defines the rules for downloading and using the GreggularVST plugin, as well as personal data processing rules for the download and contact forms.

  1. The plugin is an R&D release and may contain defects.
  2. The user uses the plugin at their own responsibility.
  3. The download link and SHA256 checksum are provided after successful form submission.
  4. Using the tool for unlawful activities is prohibited.

  5. The administrator may temporarily limit download access to maintain system security and protect against abuse.
