VST Plugin

GreggularVST

GreggularVST is a custom JUCE plugin (VST3) for generative processing of short recordings and building multilayer granular structures. The tool supports composition work, sound design, and experimental approaches to sound creation.

Interface panels

Panels

Core creation layer and key granular parameters.

Active panel

Controls Panel

Core creation layer: Duration, Pitch, Gain, Envelope, and Cloud. This is where parameters that shape the character and dynamics of the granular cloud are configured.

Panel details

  • Range and dynamics control
  • Fast access to key parameters
  • Iteration-focused working mode

Research context and conference materials are available in the Research tab.
