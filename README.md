# Scholarship Project Website

Bilingual Astro website (PL/EN) documenting the scholarship project:

`Nowe technologie w rozwoju narzedzi i kompetencji technologicznych dla wspolczesnej muzyki`

## Stack
- Astro
- Tailwind CSS (via Vite plugin)
- MDX support enabled

## Local development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Main routes
- `/pl/` and `/en/`
- `/pl/project/` and `/en/project/`
- `/pl/downloads/` and `/en/downloads/`
- `/pl/updates/` and `/en/updates/`
- `/pl/bio/` and `/en/bio/`

## Progress tracking
- `docs/tasks/PROJECT_PROGRESS.md`
