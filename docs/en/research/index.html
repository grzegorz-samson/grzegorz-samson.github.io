Audio Developer Conference 2025

The project was presented during Audio Developer Conference 2025 (November 10-12, 2025, Bristol, UK and online).

The presentation focused on combining text-prompt layers with generative audio processing and a real-time granular VST engine.

  • Poster topic: "From Text Wishes to Sound".
  • Scope: text pipeline -> audio generation -> granular processing -> resulting texture.
  • Focus: practical use of AI in creative and educational workflows.

Licensing terms of the original source code are published on the source page: https://osf.io/9m5dh/.
